

## 일러두기

* 본 기고문은 텍스트파일인 Markdown으로 작성되었습니다.

* 모든 Markdown 파일은 pdf, docx 파일로도 변환해 두었습니다.

* 모든 글을 웹브라우저 상에서 편하게 볼 수 있도록, Mdwiki도 함께 동봉해 두었습니다.  index.html 파일을 웹브라우저로 열어 보면 됩니다.



## 모질라 파이어폭스에서 index.html을 열어보는 방법

* 그냥 열어보면 됩니다.



## 윈도에서 구글 크롬으로 index.html을 열어보는 방법

* 구글 크롬 실행 아이콘에서 마우스 오른쪽 버튼을 누르시고,

* 구글 크롬 속성 입력 창이 뜨면, '바로가기' 탭에서 '대상(T)' 항목의 내용에서, 다음 '--allow-file-access-from-files' 옵션 구문을 마지막에 덧붙여 줍니다.

* 그리고 구글 크롬을 모두 껐다가, 이 실행 아이콘을 더블클릭해서 다시 실행합니다.

* 이제 index.html 파일을 크롬으로 던져넣어 열어 보면 됩니다.
